float속성은 복잡한 형태의 레이아웃을 구성하는데 필요한 핵심속성

특정 요소를 떠있게, 흐르도록, 부유하게 하도록 하는 속성

이 속성을 사용해 왼쪽, 또는 오른쪽으로 부유시키는 레이아웃 기법

float 속성을 이용하면 문서의 흐름과 관계 없이 화면 배치를 유연하게 할 수 있음.
