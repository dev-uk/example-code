position

position은 HTML 요소가 페이지에서 보여질 위치를 지정하는 속성

top bottom left right 속성을 사용하여 배치

position를 설정해야 속성이 작동함.

static 이외의 다른 값들은 어떤 기준에 의해 요소의 상대적인 위치를 설정함

- relative(상대 위치) : 요소의 기준 위치에서 속성 값을 지정하여 위치

- fixed(고정 위치) : 뷰포트(viewport)에 상대적으로 위치

- absolute(절대 위치) : 위치가 설정된 바로 위의 조상요소에 상대적으로 위치

Position 속성

bottom : 요소의 아래에 여백을 설정함

top : 요소의 위에 여백을 설정함

left : 요소의 왼쪽에 여백을 설정함

right : 요소의 오른쪽에 여백을 설정함

z-index : 요소의 스택 순서를 설정함

clip : absolute의 속성값을 가진 요소를 자름
