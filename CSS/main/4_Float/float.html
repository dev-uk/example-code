Float & Clear

float 속성은 요소가 부동상태가 되면서 문서의 보통 흐름에서 빠짐

clear 속성은 floating 된 요소에 의해 영향을 받지 않도록 설정함.

float : none

내용

none 속성값은 기본값이며, 요소가 floating 되지 않음.
