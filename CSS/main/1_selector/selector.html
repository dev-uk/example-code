내용

요소의 id는 페이지 내에서 고유하므로 하나의 요소를 선택하는 데 사용

그룹 선택자

내용

그룹 선택자는 여러 선택자를 같이 사용하여 선택

여러 선택자를 쉼표(,)로 구분하여 사용

여러 요소에 같은 스타일을 줄때 사용하여 코드의 중복을 줄여줌

태그명,클래스명,..을 사용하여 선택

예) .class,h1,h2 {...}

Code

동위 선택자

동위 선택자는 동위 관계에서 뒤에 위치한 태그를 선택

* 동위 관계 : 부모 요소가 같은 요소들

일반 동위 선택자

일반 동위 선택자는 동위 관계의 요소중 해당 요소보다 뒤에 존재하는 특정 타입의 요소를 선택

태그명~태그명을 사용하여 선택

예) div ~ p {...}
