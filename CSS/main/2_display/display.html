display

display 속성은 페이지 내에 요소가 어떻게 보일 것인가를 지정하는 중요한 속성

블록(block)

내용

display 속성값이 block인 요소는 언제나 새로운 라인에서 시작하며, 해당 라인의 모든 너비를 차지함

width height 속성 지정 가능

HTML 블럭요소 : div h1~h6 p ul ol form article aside dd dl header ...

인라인(inline)

내용

display 속성값이 inline인 요소는 새로운 라인에서 시작하지않음.

요소의 너비도 요소의 내용 만큼만 차지함

margin은 top, bottom은 적용 불가

width height 속성 지정 불가

HTML 인라인요소 : span a img b input textarea label button ...

인라인-블록(inline-block)

내용

display 속성값이 inline-block인 요소는 인라인 요소처럼 동작하지만 해당 요소 내부에서는 블록 요소처럼 동작함
